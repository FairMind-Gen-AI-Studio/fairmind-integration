The Skills Framework

fairmind-context

Foundation: Gathers comprehensive context

  • Project and session identification
  • Task plan, user story, requirements
  • Test expectations and RAG documentation

Used by all other skills

fairmind-tdd

Development: TDD aligned with criteria

  • RED: Write test from Fairmind criteria
  • GREEN: Implement following plan
  • REFACTOR: Clean code
  • DOCUMENT: Update journal

Requires: fairmind-context

fairmind-code-review

Validation: Three-layer verification

  • Layer 1: Plan vs Journal
  • Layer 2: Requirements check
  • Layer 3: Cross-repo validation
  • Output: Review report

Requires: fairmind-context

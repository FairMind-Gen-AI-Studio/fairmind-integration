fairmind-context Skill

Foundation: Comprehensive Context Gathering

What It Does

Raccoglie TUTTO il contesto necessario prima di iniziare qualsiasi lavoro su Fairmind. È la base per le altre skill.

8-Step Process

  • 1. Identify project
  • 2. Identify active session
  • 3. Gather task (implementation plan)
  • 4. Gather user story (acceptance criteria)
  • 5. Gather requirements (functional + technical)
  • 6. Gather test expectations
  • 7. Gather documentation via RAG
  • 8. Cross-project context (if integrations)

Output: Structured document with all gathered info

Why It Matters

  • Prevents working with incomplete information
  • Ensures alignment with business requirements
  • Identifies integration points early
  • Foundation for fairmind-tdd and fairmind-code-review

Key Benefits

  • No guessing: All info from Fairmind sources
  • Complete picture: Task, story, requirements, tests
  • Cross-project support: Understands integrations
  • Reusable: Same process for all development work
